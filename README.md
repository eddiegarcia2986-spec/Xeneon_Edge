To ensure the Discord widget functions correctly, you must enable the widget API within your server settings.
Enabling the Widget

    Open Discord and click your Server Name (top left) to open the dropdown menu.

    Select Server Settings.

    In the left sidebar, look for the Engagement category.

    Select Server Widget.

    Toggle Enable Server Widget to ON.

How to Configure "Quick Channels"

To make the navigation buttons jump to specific chats within your server:

    In Discord, Right-Click on a channel name (e.g., #general).

    Select Copy Link.

    Open the widget code and find the section for buttons.

    Paste that link into the href attribute where it says: href="https://discord.com/channels/245428635006271490/CHANGEME"
